7. États Interactifs

Hover

  • Boutons: Changement de gradient ou opacité
  • Liens: Changement de couleur vers #60A5FA
  • Cartes: Augmentation de la bordure (border-white/30)
  • Icônes: Scale transform (scale-110)

Focus

  • Inputs: Bordure bleue avec ombre
  • Boutons: Même style que hover
  • Outline: Supprimé, remplacé par box-shadow

Active

  • Boutons: Gradient plus foncé
  • Navigation: Background bleu transparent

8. Responsive Design

Breakpoints

  • Mobile: < 768px
  • Tablet: 768px - 1024px
  • Desktop: > 1024px
  • Large Desktop: > 1440px

Principes

  • Mobile-first approach
  • Grilles adaptatives (grid-cols-1 md:grid-cols-2)
  • Typographie responsive (text-4xl md:text-6xl)
  • Espacements adaptatifs (px-6 md:px-12 lg:px-24)

9. Animations

Transitions

  • Durée standard: 0.3s
  • Easing: ease-in-out (par défaut)
  • Propriétés animées: all, transform, opacity, colors

Animations CSS

  • Pulse: Pour les éléments décoratifs
  • Fade-in: Pour les apparitions
  • Scale: Pour les interactions hover

10. Règles d'Utilisation

Couleurs

  • Utiliser les gradients avec modération, principalement pour les CTA
  • Respecter la hiérarchie des opacités pour le texte
  • Ne jamais utiliser de texte blanc pur sur fond blanc
  • Les couleurs d'accent (bleu/violet) doivent être utilisées pour attirer l'attention

Typographie

  • Respecter la hiérarchie des tailles
  • Utiliser les poids de police appropriés
  • Maintenir une bonne lisibilité (contraste suffisant)

Composants

  • Toujours appliquer l'effet glassmorphism sur les cartes
  • Utiliser les ombres colorées pour la profondeur
  • Respecter les espacements cohérents
  • Maintenir la cohérence visuelle entre les pages
